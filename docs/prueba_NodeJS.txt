La prueba consiste en desarrollador un backend restful con NodeJs y Express para un crud de usuarios. Como apoyo se utilizo el swagger proporcionado.

Dentro de la carpeta "models" se encuentran definido 3 modelos:
    1. El de servidor en el cual se encnuentra la instancia del app, se definen los middlewares necesarios, la ruta base para el crud del usuario y un metodo para iniciar el servidor.
    2. El del usuario el cual es el modelo de la tabla en la base de datos creado en base al swagger proporcionado.
    3. El de direccion el cual es modelo de la tabla en la base de datos creado en base al swagger proporcionado y tambien se encuentra creada la relacion entre la tabla de usuario.

Dentro de la carpeta "config" se encuentra la coniguracion y conexion a la base de datos creado con Sequelize.
Dentro de la carpeta "controller" se encuentran un archivo js con los que es el controlador del modelo de usuarios para procesar los datos recibidos y enviados de cada enpoint correspondiente.
Dentro de la carpeta "middlewares" se encuentra la logica para hacer validaciones necesarias para tener un mejor contro los errores de cada enpoint.
Dentro de la carpeta "routes" se encuentra el archivo js con las rutas de todos los enpoints para usuarios, dentro de la cual se hace uso de los middlewares y los controladores respectivamente.
Dentro de la carpeta "tests" se encuentra el archivo con los test correspondientes a los enpoints de usuario.

Se hizo uso de las variables de entorno para la seguirdad de credenciales, en este caso la de la base de datos.
